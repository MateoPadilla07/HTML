HTML

HTML son las siglas de Hyper Text Markup Languaje que puede traducirse como lenguaje de marcas o marcada de hipertexto.

El lenguaje HTML se emplea para crear paginas web. Es muy facil ver el codigo HTML de una pagina web. la opcion exacta cambia de un navegador a otro y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener un nimbre similar

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-lee creo la primera pagina web.
